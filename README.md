# Gillow | Patent & IP Marketplace

Gillow is a high-fidelity, Zillow-inspired technology and patent intelligence platform. It allows users to browse, analyze, and track intellectual property assets with a focus on visual data, map-based discovery, and deep-link filtering.

## 🚀 Features

- **Zillow-Inspired UI**: Familiar property-browsing patterns applied to patent assets.
- **Interactive Landscapes**: Map-based clustering of patents by assignee and technology.
- **Advanced Filtering**: Search by valuation, citation thresholds, claims count, and legal status.
- **Valuation Intelligence**: AI-driven price estimates and historical valuation tracking.
- **Collection Management**: Save favorites, organize searches, and compare assets side-by-side.
- **Technical Detail View**: Interactive claims trees, citation trends, and image galleries.

## 🛠 Tech Stack

- **React 19**: Modern component-based architecture.
- **TypeScript**: Full type safety for complex IP data structures.
- **Tailwind CSS**: Utility-first styling with custom animation extensions.
- **Lucide React**: Clean, consistent iconography.
- **Recharts**: Data visualization for valuation and citation trends.
- **React Router 7**: Robust client-side navigation and deep linking.

## 🏃 How to Run Locally

1. Ensure you have Node.js installed.
2. Clone this project.
3. Run `npm install` to install dependencies.
4. Run `npm run dev` to start the development server.
5. Open `http://localhost:5173` (or your local port) in your browser.
